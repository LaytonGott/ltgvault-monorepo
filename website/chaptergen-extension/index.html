ChapterGen Chrome Extension

Automatically grab transcripts from any YouTube video with one click.

Installation (Developer Mode)

1

Download the extension

Click the button below to download the extension files as a ZIP.

2

Unzip the folder

Extract the ZIP file to a location on your computer.

3

Open Chrome Extensions

Go to chrome://extensions in your browser.

4

Enable Developer Mode

Toggle "Developer mode" in the top right corner.

5

Load the extension

Click "Load unpacked" and select the unzipped folder.

Download Extension
How to use: Navigate to any YouTube video, click the ChapterGen extension icon, then click "Grab Transcript". The transcript will be extracted and you can send it directly to ChapterGen with one click.
